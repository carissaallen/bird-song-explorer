# Western Meadowlark Songs

Place bird song audio files here:

- `song_01.mp3` - Primary territorial song
- `song_02.mp3` - Alternative song variation
- `call_01.mp3` - Contact call
- `call_02.mp3` - Flight call
- `alarm_01.mp3` - Alarm/warning call

## Song Guidelines

- Format: MP3 (128-320 kbps)
- Duration: 10-30 seconds per file
- Quality: Clear recording with minimal background noise
- Normalize audio levels for consistency
